# 1113354_Poker

這不只是一個撲克牌遊戲，這是一個整合了最新科技趨勢的數位娛樂平台。

## 三大核心技術

### 影像處理與電腦視覺 (Image Processing & Computer Vision)
- **AI 視覺化分析面板**：發牌後可開啟專屬的 AI 分析介面，將每項技術處理的特徵以視覺化的方式呈現在牌面影像上。
- **暗通道先驗 (Dark Channel Prior)**：手動對比度與去霧分析，增強影像。
- **梯度直方 (HOG)**：用於分析與提取牌面特徵邊緣輪廓。
- **遮罩 (Mask R-CNN)**：透過像素級分割對撲克牌建立遮罩，區隔背景與主體物件。
- **多目標追蹤 (DeepSORT)**：追蹤發牌與抽換過程的順序軌跡。
- **物件偵測 (R-CNN)**：自動標記每張撲克牌方位並回饋 Confidence Score。
- **光學字元辨識 (OCR)**：即時讀取並識別花色與點數資訊。

### 響應式介面設計 (Responsive UI)
- 完美支援全螢幕與動態視窗縮放。撲克牌及所有 UI 控制項都會根據可用版面等比例擴展與置中。

### 深度學習 (Deep Learning)
- **智能勝率預測模型**：動態展示運算後的勝率百分比，並且已經實作專屬 UI 標籤顯示於主介面下方，給予最清爽明瞭的輔助參考。

### 資安防護 (InfoSec / CIA Triad)
- **機密性 (Confidentiality)**：所有的資金變動都將進行 Base64/Hash 加密轉換。
- **完整性 (Integrity)**：確保資金在變動過程中未被惡意竄改。
- **可用性 (Availability)**：嚴謹的 Exception Handling 防呆機制，維持系統的可用性。
